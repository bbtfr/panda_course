欢迎来到 PANDA 2014 研讨会

本次研讨会由山东大学计算机科学与技术学院、Open Security Research和山东华芯半导体有限公司共同举办。旨在为国内外密码工程领域的在校学生、学者以及业界专家搭建一个交流平台。PANDA专注于密码芯片、密码设备的侧信道攻击和安全测评。
